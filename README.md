# AI Driven Cyber Threat Intelligence

## Project Overview
This project focuses on developing an AI-driven Cyber Threat Intelligence (CTI) system
using machine learning techniques to detect and analyze network intrusions.

## Dataset
The project uses the NSL-KDD dataset for training and evaluation of intrusion detection models.

## Project Structure
- data/        : Raw and processed datasets
- notebooks/  : Experiments and analysis
- src/         : Source code (preprocessing, training, evaluation)
- models/      : Trained models (not tracked in Git)
- reports/     : Results and evaluation reports
- logs/        : Execution logs

## Tools & Technologies
- Python
- Scikit-learn
- Pandas
- NumPy
